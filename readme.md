Frontend Mentor - Weather App Solution

This is a complete solution for the Weather app challenge on Frontend Mentor, built to provide real-time, responsive weather data.



The application meets all challenge requirements, including:

Current Weather & Details: Displays temperature, location, date, description, "feels like," humidity, wind, and precipitation.

Forecasts: Implements both a 7-day daily forecast and a detailed hourly forecast.

Search & API: Utilizes city search (via Nominatim) and fetches live data from the Open-Meteo API.

Unit Toggling: Allows switching between Metric (°C, km/h) and Imperial (°F, mph) units.

Design: Full dark-mode responsiveness based on the provided style guide.
